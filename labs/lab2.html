Lab 1: The Artemis

Objective

The objective of this lab is to establish Bluetooth communication between the Artemis and the computer. We need to implement Arduino code on the Artemis side and Python in Jupyter Notebook on the computer. The purpose of this is to offload computation to the computer later when we are performing computationally intensive tasks on the robot.

Materials and Tools Used

  • 1 Sparkfun Artemis RedBoard Nano
  • 1 USB-C to USB-C Cable
  • bluetooth receiver

Procedure

  1. Setup


    Python virtual environment and jupyter lab setup.

  2. Bluetooth Connection


    To properly establish a connnection to the artemis nano, we need to obtain the MAC address of the Artemis and edit the connection.yaml to reflect this address.

  3. Serial


    We then upload the serial example to check the serial port works. To check the output and provide input open the serial monitor in the upper right hand corner of the script window.

  4. Analog Read (Temperature)


    Next, the Analog read example was uploaded. This program measures the temperature using the board's onboard sensor. A temperature change was reprted when the board was held in the hand. The Video below shows the change in teperature from 85F to 90F.

  5. Microphone


    We then upload the MicrophoneOutput example to test the inbult microphone. The program reports the frequency of the loudest sound. We can change the frequency reported by speaking or making/playing a sound next to the Artemis Nano. We can see that the frequency of the loudest sound changes over time in response to the microphone input.

  6. LED blink with Microphone


    Next, We create a program where the LED lights up in response to sound heard b the onboard microphone. Here, the LED lights up when the frequency of the sound is greater than 2000Hz.
