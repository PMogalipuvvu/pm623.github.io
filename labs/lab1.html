Lab 1: The Artemis

Objective

The purpose of this lab is to setup and become familiar with the Arduino IDE and the Artemis RedBoard Nano from Sparkfun. In this lab, We program the board, blink the LED, read/write serial messages over USB, display the output from the onboard temperature sensor, and measure the loudest frequency recorded by the Pulse Density Microphone.

Materials and Tools Used

  • 1 Sparkfun Artemis RedBoard Nano
  • 1 USB-C to USB-C Cable

Procedure

  1. Arduino IDE and Apollo3 board package was installed to allow the Artemis RedBoard Nano to be compatible. Artemis RedBoard Nano was connected to the computer via the USB-C to USB-C Cable.
  2. Next, the Blink example was uploaded to the board to make the built-in LED bilnk. The LED blinks repeatedly with an interval of 1s. The video below demostrates the program
  3. We then upload the serial example to check the serial port works. To check the output and provide input open the serial monitor in the upper right hand corner of the script window.
  4. Next, the Analog read example was uploaded. This program measures the temperature using the board's onboard sensor. A temperature change was reprted when the board was held in the hand. The Video below shows the change in teperature from 85F to 90F.
